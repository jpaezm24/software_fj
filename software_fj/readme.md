# Sistema Integral Software FJ

Proyecto desarrollado en Python usando Programación Orientada a Objetos.

## Características

- Manejo de clientes
- Gestión de servicios
- Sistema de reservas
- Manejo avanzado de excepciones
- Herencia
- Polimorfismo
- Encapsulación
- Registro de logs

## Servicios implementados

1. Reserva de salas
2. Alquiler de equipos
3. Asesorías especializadas

## Tecnologías usadas

- Python 3
- Programación Orientada a Objetos
- GitHub
